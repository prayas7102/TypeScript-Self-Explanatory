## Run Locally

Clone the project

```bash
  git clone https://github.com/prayas7102/TypeScript-Self-Explanatory
```

Go to the root directory to run the TypeScript code: 

```bash
  tsc -w app.ts
```
OR

Go to the project directory to run the todo list (react)

```bash
  cd main_project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
